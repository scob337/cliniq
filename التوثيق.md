# 📋 توثيق سيرفر كلينيك الطبي

## 🎯 نظرة عامة

هذا السيرفر يوفر واجهات برمجية (API) لخدمة البيانات الطبية الثابتة. تم تنظيم البيانات بشكل منظم وسهل التعديل والإضافة.

---

```
lib/data/
├── specialties.ts    # 43 تخصص طبي
├── provinces.ts      # 8 محافظات مع مناطقها
├── services.ts       # 8 خدمات طبية
└── doctors.ts        # 12 طبيب وعيادة ومستشفى
```

---

## 🔌 نقاط النهاية (API Endpoints)

### 1️⃣ التخصصات الطبية
**نقطة النهاية:** `GET /api/specialties`

تحصل على قائمة جميع التخصصات الطبية مع الصور والأوصاف

**مثال الاستجابة:**

```json
{
  "نجح": true,
  "البيانات": [
    {
      "id": "dermatology",
      "name": "جلدية",
      "description": "أمراض الجلد والعلاج الجمالي",
      "image": "https://via.placeholder.com/150?text=جلدية"
    }
  ],
  "العدد": 43
}
```

---

### 2️⃣ المحافظات والمناطق
**نقطة النهاية:** `GET /api/provinces`

**المعاملات (Parameters):**
- `محافظة` - معرف المحافظة (اختياري)

**مثال الاستجابة:**

```json
{
  "نجح": true,
  "البيانات": [
    {
      "id": "cairo",
      "name": "القاهرة",
      "areas": [
        {
          "id": "cairo_downtown",
          "name": "وسط البلد"
        }
      ]
    }
  ],
  "العدد": 8
}
```

**أمثلة الاستخدام:**
- احصل على مناطق القاهرة فقط:
  ```
  GET /api/provinces?محافظة=cairo
  ```

---

### 3️⃣ الخدمات الطبية
**نقطة النهاية:** `GET /api/services`

تحصل على قائمة الخدمات الطبية المتوفرة

**مثال الاستجابة:**

```json
{
  "نجح": true,
  "البيانات": [
    {
      "id": "consultation",
      "name": "الاستشارات الطبية",
      "description": "استشارات طبية من أطباء متخصصين",
      "image": "https://via.placeholder.com/150?text=استشارات"
    }
  ],
  "العدد": 8
}
```

---

### 4️⃣ الأطباء والعيادات والمستشفيات
**نقطة النهاية:** `GET /api/doctors`

**المعاملات (Parameters):**
- `معرف` - معرف الطبيب (اختياري)
- `تخصص` - التخصص (اختياري)
- `محافظة` - المحافظة (اختياري)
- `منطقة` - المنطقة (اختياري مع محافظة)
- `بحث` - البحث عن نص (اختياري)

**مثال الاستجابة:**

```json
{
  "نجح": true,
  "البيانات": [
    {
      "id": "doc_001",
      "name": "د. أحمد محمد علي",
      "specialty": "dermatology",
      "experience": 15,
      "rating": 4.8,
      "reviews": 142,
      "province": "cairo",
      "area": "cairo_downtown",
      "phone": "+201001234567",
      "email": "ahmed@clinic.com",
      "type": "doctor",
      "image": "https://via.placeholder.com/200?text=د.احمد",
      "qualifications": [
        "بكالوريوس الطب - جامعة القاهرة",
        "ماجستير الأمراض الجلدية"
      ],
      "services": ["consultation", "medical_services"],
      "about": "متخصص في أمراض الجلد والعلاجات التجميلية بخبرة 15 سنة"
    }
  ],
  "العدد": 1
}
```

**أمثلة الاستخدام:**
- احصل على طبيب بمعرف محدد:
  ```
  GET /api/doctors?معرف=doc_001
  ```

- احصل على جميع أطباء جلدية:
  ```
  GET /api/doctors?تخصص=dermatology
  ```

- احصل على الأطباء في القاهرة:
  ```
  GET /api/doctors?محافظة=cairo
  ```

- احصل على الأطباء في منطقة وسط البلد:
  ```
  GET /api/doctors?محافظة=cairo&منطقة=cairo_downtown
  ```

- ابحث عن طبيب أو عيادة:
  ```
  GET /api/doctors?بحث=أحمد
  ```

## 📝 كيفية إضافة وتعديل البيانات

### إضافة تخصص جديد
افتح ملف `lib/data/specialties.ts` وأضف:

```typescript
{
```

## 🚀 الاستخدام من الواجهة الأمامية

### استخدام React Query

```typescript
import { useQuery } from '@tanstack/react-query';

// احصل على التخصصات
const { data: specialties } = useQuery({
  queryKey: ['specialties'],
  queryFn: () => fetch('/api/specialties').then(r => r.json())
});

// احصل على أطباء التخصص
const { data: doctors } = useQuery({
  queryKey: ['doctors', specialty],
  queryFn: () => fetch(`/api/doctors?تخصص=${specialty}`).then(r => r.json())
});
```

### استخدام Fetch API

```typescript
// احصل على المحافظات
const response = await fetch('/api/provinces');
const { البيانات } = await response.json();

// ابحث عن أطباء
const searchResponse = await fetch(`/api/doctors?بحث=أحمد`);
const { البيانات: searchResults } = await searchResponse.json();
```
