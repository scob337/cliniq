## 🎓 أمثلة الاستخدام

### JavaScript/TypeScript:
```typescript
// جلب التخصصات مع فلترة
const response = await fetch(
  '/api/specialties?province=cairo&maxPrice=400'
);
const data = await response.json();
console.log(data.البيانات); // التخصصات المرشحة
```

### React:
```tsx
const [specialties, setSpecialties] = useState([]);

useEffect(() => {
  fetch('/api/specialties?province=cairo')
    .then(res => res.json())
    .then(data => setSpecialties(data.البيانات));
}, []);

return (
  <div>
    {specialties.map(s => (
      <div key={s.id}>
        <h2>{s.name}</h2>
        <p>{s.consultation_fee} جنيه</p>
      </div>
    ))}
  </div>
);
```

### PowerShell:
```powershell
Invoke-WebRequest -Uri "http://localhost:3000/api/specialties?province=cairo" | 
  ConvertFrom-Json | 
  Select-Object -ExpandProperty البيانات
```
