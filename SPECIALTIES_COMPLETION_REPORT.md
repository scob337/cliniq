## 🚀 ميزات الـ API الجديدة

### معاملات الفلترة:

```
GET /api/specialties?province=cairo&minPrice=300&maxPrice=450&search=جراحة
```

| المعامل | النوع | الوصف | مثال |
|--------|------|-------|------|
| `province` | string | فلترة حسب المحافظة | `cairo` |
| `minPrice` | number | السعر الأدنى | `250` |
| `maxPrice` | number | السعر الأقصى | `400` |
| `search` | string | بحث في الاسم/الوصف | `جراحة` |

### استجابة الـ API:

```json
{
  "نجح": true,
  "البيانات": [
    {
      "id": "dermatology",
      "name": "جلدية",
      "description": "أمراض الجلد والعلاج الجمالي والليزر",
      "image": "URL",
      "consultation_fee": 350,
      "provinces": ["cairo", "giza", "beheira"],
      "doctors_count": 8
    }
  ],
  "العدد": 2,
  "الإجمالي": 43,
  "الفلاتر": {
    "المحافظة": "cairo",
    "السعر_من": "300",
    "السعر_الى": "450",
    "البحث": "جراحة"
  }
}
```
