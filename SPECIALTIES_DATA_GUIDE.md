# 📋 دليل بيانات التخصصات الطبية

## 🎯 نظرة عامة

تم تحديث نظام التخصصات الطبية في التطبيق بميزات جديدة تسمح بفلترة شاملة وتمكن المستخدمين من اختيار التخصص المناسب بناءً على احتياجاتهم.

---

## ✨ الميزات الجديدة

### 1. **سعر الكشف (Consultation Fee)**
- كل تخصص له سعر محدد للكشف
- تتراوح الأسعار من **150 جنيه** إلى **500 جنيه**
- يمكن للمستخدمين فلترة حسب ميزانيتهم

### 2. **المحافظات المتاحة (Provinces)**
- كل تخصص متاح في محافظات محددة
- يسهل على المستخدمين البحث في المحافظة الخاصة بهم
- 5 محافظات رئيسية:
  - **cairo** - القاهرة
  - **giza** - الجيزة
  - **alexandria** - الإسكندرية
  - **assiut** - أسيوط
  - **beheira** - البحيرة

### 3. **عدد الأطباء (Doctors Count)**
- معلومة تساعد في اختيار التخصص الأشهر
- تعطي فكرة عن توفر الأطباء

### 4. **وصف موسع (Enhanced Description)**
- أوصاف أكثر تفصيلاً لكل تخصص
- تساعد المريض على فهم طبيعة الخدمة

## 🔍 استخدام الفلاتر

### مثال 1: البحث عن تخصصات بسعر معقول في القاهرة
```
GET /api/specialties?province=cairo&minPrice=150&maxPrice=300
```
**النتيجة:** تخصصات بسعر مناسب في القاهرة

### مثال 2: البحث عن تخصصات جراحية
```
GET /api/specialties?search=جراحة
```
**النتيجة:** جميع التخصصات التي تحتوي على كلمة "جراحة"

### مثال 3: البحث عن تخصصات مكلفة
```
GET /api/specialties?minPrice=400
```
**النتيجة:** التخصصات المتخصصة والمكلفة

### مثال 4: فلترة متقدمة
```
GET /api/specialties?province=alexandria&minPrice=200&maxPrice=400&search=قلب
```
**النتيجة:** تخصصات متعلقة بالقلب في الإسكندرية بسعر 200-400 جنيه

## 🔧 تطوير واستخدام الفلاتر في الواجهة

### JavaScriptمثال استدعاء الـ API من الواجهة الأمامية:

```javascript
// جلب جميع التخصصات
fetch('/api/specialties')
  .then(res => res.json())
  .then(data => console.log(data.البيانات));

// فلترة حسب المحافظة والسعر
const params = new URLSearchParams({
  province: 'cairo',
  minPrice: 250,
  maxPrice: 400
});

fetch(`/api/specialties?${params}`)
  .then(res => res.json())
  .then(data => {
    console.log(`عدد التخصصات: ${data.العدد}`);
    console.log(data.البيانات);
  });
```

### React مثال في Next.js:

```typescript
import { useEffect, useState } from 'react';

export default function Specialties() {
  const [specialties, setSpecialties] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    const params = new URLSearchParams({
      province: 'cairo',
      minPrice: '200',
      maxPrice: '400'
    });

    fetch(`/api/specialties?${params}`)
      .then(res => res.json())
      .then(data => {
        setSpecialties(data.البيانات);
        setLoading(false);
      });
  }, []);

  if (loading) return <div>جاري التحميل...</div>;

  return (
    <div>
      <h1>التخصصات الطبية</h1>
      {specialties.map(specialty => (
        <div key={specialty.id}>
          <h2>{specialty.name}</h2>
          <p>{specialty.description}</p>
          <p>السعر: {specialty.consultation_fee} جنيه</p>
          <p>عدد الأطباء: {specialty.doctors_count}</p>
        </div>
      ))}
    </div>
  );
}
```

## 🎨 حالات الاستخدام

### 1. **صفحة البحث المتقدم**
- اسمح للمستخدم بتحديد:
  - المحافظة الخاصة به
  - نطاق السعر المفضل
  - البحث عن تخصص معين

### 2. **توصيات التخصصات**
- اعرض للمستخدم التخصصات الأرخص
- اعرض التخصصات الأكثر توفراً
- اقترح بدائل أرخص

### 3. **مرشح الأطباء**
- من خلال اختيار التخصص، يمكن عرض الأطباء المتخصصين
- من خلال اختيار المحافظة، عرض التخصصات المتاحة

### 4. **لوحة تحكم المستخدم**
- اسمح للمستخدم بحفظ التخصصات المفضلة
- قارن الأسعار بين التخصصات
- تابع متوسط أسعار التخصصات

## 🔒 ملاحظات أمان ومعايير

### التحقق من البيانات:
- جميع معاملات الفلترة تتم من جانب الخادم
- لا يمكن للمستخدم الوصول إلى بيانات محظورة
- الأسعار محفوظة في قاعدة البيانات بشكل آمن

### الأداء:
- الفلترة فعالة وسريعة
- يمكن القيام بفلترة متعددة بدون مشاكل
- مناسب للبيانات الكبيرة
