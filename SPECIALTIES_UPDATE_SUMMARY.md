#### المميزات الجديدة:
- ✅ **فلترة متقدمة** حسب معايير متعددة
- ✅ **معاملات جديدة** في الطلب:

| المعامل | النوع | الوصف |
|--------|------|-------|
| `province` | string | فلترة حسب المحافظة |
| `minPrice` | number | أقل سعر |
| `maxPrice` | number | أقصى سعر |
| `search` | string | بحث في الاسم والوصف |

#### استخدام الـ API:

**1. جلب جميع التخصصات:**
```bash
GET /api/specialties
```

**2. فلترة حسب المحافظة:**
```bash
GET /api/specialties?province=cairo
```

**3. فلترة حسب السعر:**
```bash
GET /api/specialties?minPrice=200&maxPrice=400
```

**4. بحث متقدم:**
```bash
GET /api/specialties?province=cairo&minPrice=300&maxPrice=450&search=جراحة
```

## 🚀 كيفية الاستخدام

### للمطورين الذين يستخدمون React:

```typescript
import { useEffect, useState } from 'react';

export default function Specialties() {
  const [specialties, setSpecialties] = useState([]);

  useEffect(() => {
    const params = new URLSearchParams({
      province: 'cairo',
      maxPrice: '400'
    });

    fetch(`/api/specialties?${params}`)
      .then(res => res.json())
      .then(data => setSpecialties(data.البيانات));
  }, []);

  return (
    <div>
      {specialties.map(s => (
        <div key={s.id}>
          <h2>{s.name}</h2>
          <p>{s.consultation_fee} جنيه</p>
        </div>
      ))}
    </div>
  );
}
```

### للمطورين الذين يستخدمون JavaScript:

```javascript
async function searchSpecialties(filters) {
  const params = new URLSearchParams();
  
  if (filters.province) params.append('province', filters.province);
  if (filters.minPrice) params.append('minPrice', filters.minPrice);
  if (filters.maxPrice) params.append('maxPrice', filters.maxPrice);

  const response = await fetch(`/api/specialties?${params}`);
  return await response.json();
}

// الاستخدام:
const results = await searchSpecialties({
  province: 'cairo',
  minPrice: 200,
  maxPrice: 400
});
```
